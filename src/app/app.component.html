Saving Glaciers
Waste management is a serious concern as waste segregation and collection practices are not followed correctly. This project aims to address this issue by providing civic bodies in charge of waste management with the appropriate data for Efficient Data collection and segregation. This data can be later aggregated at a block/area level allowing civic bodies to concentrate waste collection and segregation efforts in a particular place.
